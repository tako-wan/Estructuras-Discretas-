# Práctica3

## Int:
Int representa un tipo  concreto de dato que almacena especificamente valores/números de tipo entero

##Num:
A diferencia de Int, Num puede almacenar cualquier valor de tipo númerico.

Ejemplo:

    ghci> :t 5
    5 :: Num p => p
    ghci> :t (5 :: Int)
    5 :: Int
    ghci> :info Num
